Data Mules

The field of wireless sensor networks has grown in recent years due to it's ability to measure interesting phenomenon . In some applications though it is impractical to completely cover the area with sensors, in these situations robots can act as intermediaries carrying the data between sensors. In this project we worked to answer fundamental questions about these hybrid robot sensor networks.

In what order should the robot visit the static nodes? This problem differs from the nominal TSP problem because the robot must take into account the time it takes at the node to download the data as well as the travel time.

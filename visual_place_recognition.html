Visual Place Recognition

How do you know when you are? When people walk up to their house they don't recognize their location solely because of the number next to the door: the size of the door, color of the wall, and distance between the front windows. We capitalize on these relationships to build a description of each image, which we use to state how similar two images are. All of these factors allow us to run place recognition and place re-recognition on limited computing devices such as mobile phones and embedded devices in real time.
